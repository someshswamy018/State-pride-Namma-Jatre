# State Pride – Namma Jatre

## 📌 Project Overview

State Pride – Namma Jatre is a smart digital guide platform developed for village fairs and cultural events.

The application helps visitors, volunteers, and organizers manage events efficiently through real-time schedules, Lost & Found services, emergency support, cultural stories, and admin management.

---

## 🚀 Features

- Live Event Schedules
- Lost & Found Management
- Emergency Support System
- Volunteer Coordination
- Parking Guidance
- Cultural Stories Section
- Admin Dashboard
- Multi-screen Navigation
- Firebase Integration
- Responsive User Interface

## ⚙️ Installation and Setup

1. Clone the repository

```bash
git clone https://github.com/someshswamy018/State-pride-Namma-Jatre.git
```

2. Open project folder

```bash
cd State-pride-Namma-Jatre
```

3. Install dependencies

```bash
npm install
```

4. Run project

```bash
npm run dev
```
